BoardJS
Examples GitHub

Overview

BoardJS is a timeline-based animation library for building presentations, stories, and interactive ads. You describe what happens using a chain of commands, then call .go() — the library plays them back as a synchronized sequence.

  • No dependencies — pure JavaScript, CSS, and HTML
  • Declarative API using options objects — no positional guessing
  • 60 fps animations via requestAnimationFrame
  • Multi-ID targeting — apply a command to several elements at once
  • Optional SoundManager2 integration for audio

Setup

Include the library, then create a BoardJS instance pointing at a container element:

const board = new BoardJS('myContainer');

Chain commands and call .go() to start:

board
  .background('rgb(30,60,120)', { duration: 600 })
  .text('Hello, World!', { x: 0, y: 160, class: 'title center', show: 'fade', duration: 1000, wait: true })
  .delay(1500)
  .go();

Call .reset() at the start of any function that reloads the board — it clears the stage and cancels any running animations:

board.reset()
  .text('New content', { y: 100, class: 'title center', show: 'fade', duration: 800 })
  .go();

Constants

Plain strings always work. These constants exist to catch typos at development time:

// Show types
BoardJS.SHOW.APPEAR      // 'appear'
BoardJS.SHOW.FADE        // 'fade'
BoardJS.SHOW.ZOOM        // 'zoom'
BoardJS.SHOW.SPLIT       // 'split'  (text only)
BoardJS.SHOW.FROM_LEFT   // 'fromleft'
BoardJS.SHOW.FROM_RIGHT  // 'fromright'
BoardJS.SHOW.FROM_TOP    // 'fromtop'
BoardJS.SHOW.FROM_BOTTOM // 'frombottom'

// Easing types
BoardJS.EASE.LINEAR   // 'linear'
BoardJS.EASE.IN       // 'easein'
BoardJS.EASE.OUT      // 'easeout'
BoardJS.EASE.BACK_IN  // 'backin'
BoardJS.EASE.BACK_OUT // 'backout'
BoardJS.EASE.ELASTIC  // 'elastic'
BoardJS.EASE.BOUNCE   // 'bounce'

Combine multiple show types as a space-separated string: 'fade fromleft'

Commands

.text(content, opts)
Add a text element to the stage.
ParameterDefaultDescription
content—Plain text or safe HTML (<b>, <a>, <br>). Script tags and inline event handlers are stripped.
opts.x optional0Left position in px.
opts.y optional0Top position in px.
opts.class optional''CSS class(es) to apply — title, description, Button, or any custom class.
opts.show optional'appear'Entrance animation. See constants above. Combine with a space: 'fade fromtop'.
opts.duration optional400Animation duration in ms.
opts.easing optional'linear'Easing formula.
opts.id optionalautoElement ID — required if you want to reference this element later.
opts.wait optionalfalseWhen true, the timeline pauses until the entrance animation finishes.
board.text('Hello!', { x: 0, y: 80, class: 'title center', show: 'fade', duration: 800, id: 'headline', wait: true })
.image(src, opts)
Add an image element to the stage.
ParameterDefaultDescription
src—Image path.
opts.x optional0Left position in px.
opts.y optional0Top position in px.
opts.width optionalautoExplicit width in px. Omit to use the image's natural width.
opts.height optionalautoExplicit height in px.
opts.show optional'appear'Entrance animation.
opts.duration optional400Animation duration in ms.
opts.easing optional'linear'Easing formula.
opts.id optionalautoElement ID.
opts.wait optionalfalsePause timeline until animation finishes.
board.image('logo.png', { x: 270, y: 157, show: 'fade zoom', duration: 500, easing: 'backout', id: 'logo', wait: true })
.background(src, opts)
Set the stage background to a color or an image.
ParameterDefaultDescription
src—CSS color ('#ff0', 'rgb(…)', 'rgba(…)') or image path.
opts.duration optional500Fade-in duration in ms.
opts.wait optionalfalsePause timeline until background is fully visible.
board.background('rgb(20, 40, 80)', { duration: 800 })
board.background('hero.jpg', { duration: 1200, wait: true })
.delay(ms)
Pause the timeline for a fixed amount of time.
ParameterDefaultDescription
ms—Duration in milliseconds.
board.delay(2000) // pause for 2 seconds
.move(id, opts)
Animate an element to new coordinates.
ParameterDefaultDescription
id—Target element ID.
opts.x optionalunchangedNew left position in px.
opts.y optionalunchangedNew top position in px.
opts.duration optional600Animation duration in ms.
opts.easing optional'linear'Easing formula.
opts.wait optionalfalsePause timeline until move finishes.
board.move('logo', { x: 100, y: 200, duration: 800, easing: 'backout', wait: true })
.fadeIn(id, opts)  /  .fadeOut(id, opts)
Smoothly show or hide an element.
ParameterDefaultDescription
id—Target element ID.
opts.duration optional500Fade duration in ms.
opts.wait optionalfalsePause timeline until complete.
board.fadeOut('headline', { duration: 400, wait: true })
board.fadeIn('headline', { duration: 600 })
.flash(id, opts)
Blink an element by fading it in and out repeatedly.
ParameterDefaultDescription
id—Target element ID.
opts.count optional3Number of blinks, or 'always' to blink indefinitely.
opts.duration optional400Duration of each blink cycle in ms.
opts.wait optionalfalsePause timeline until all blinks finish (ignored when count: 'always').
board.flash('arrow', { count: 'always', duration: 500 })
.rotate(id, angle, opts)
Rotate an element to an absolute angle.
ParameterDefaultDescription
id—Target element ID.
angle—Target angle in degrees. Positive = clockwise.
opts.duration optional500Animation duration in ms.
opts.easing optional'linear'Easing formula.
opts.wait optionalfalsePause timeline until rotation finishes.
board.rotate('wheel', -90, { duration: 1000, easing: 'easeout', wait: true })
.zoom(id, scale, opts)
Scale an element. 1 is the original size.
ParameterDefaultDescription
id—Target element ID.
scale—Target scale factor. 2 = double size, 0.5 = half.
opts.duration optional500Animation duration in ms.
opts.easing optional'linear'Easing formula.
opts.wait optionalfalsePause timeline until scaling finishes.
board.zoom('logo', 1.5, { duration: 400, easing: 'backout' })
.style(id, css)
Apply inline CSS to an element. The string is merged into the element's existing style.
ParameterDefaultDescription
id—Target element ID.
css—Inline CSS string, e.g. 'color:red; font-size:20px'
board.style('headline', 'font-size:42px; letter-spacing:2px')
.html(id, content)
Replace the inner HTML of a text element.
ParameterDefaultDescription
id—Target element ID.
content—New HTML content.
board.html('counter', 'Score: <b>42</b>')
.remove(id)
Remove one or more elements from the stage. Pass space-separated IDs to remove multiple.
board.remove('headline subline logo')
.changeImage(id, src)
Swap the source of an image element.
ParameterDefaultDescription
id—Target image element ID.
src—New image path.
board.changeImage('avatar', 'images/user2.jpg')
.on(id, event, fn)  /  .off(id, event)
Attach or remove a DOM event listener. Multiple IDs can be space-separated.
ParameterDefaultDescription
id—Target element ID(s).
event—Event name without on prefix — e.g. 'click', 'mouseover'.
fn—Handler function. this is the element, so this.id gives the element's ID.
board.on('btn1 btn2', 'click', function() {
  board.html('info', `Clicked: ${this.id}`);
})
.hover(id, css, opts)  /  .unhover(id)
Apply CSS on mouse-over, revert on mouse-out. Supports the extra properties scale and rotate.
ParameterDefaultDescription
id—Target element ID(s).
css—CSS to apply on hover. Extras: 'scale:1.2; rotate:15'.
opts.duration optional300Transition duration in ms.
opts.easing optional'linear'Easing formula.
board.hover('btn', 'scale:1.2; border:2px solid #fff', { duration: 250, easing: 'backout' })
board.unhover('btn') // cancel hover behaviour
.tooltip(id, content)  /  .removeTooltip(id)
Attach a persistent tooltip that appears on mouseover. Remove it with .removeTooltip().
board.tooltip('icon', 'Click to expand')
board.removeTooltip('icon')
.showTooltip(id, content, opts)
Show a tooltip immediately (not on hover) for a fixed duration.
ParameterDefaultDescription
id—Target element ID.
content—Tooltip HTML content.
opts.duration optionalnullHow long to display in ms. Omit for permanent (remove manually with .removeTooltip()).
opts.wait optionalfalsePause timeline until tooltip is dismissed (only meaningful when duration is set).
board.showTooltip('arrow', 'Click here to begin', { duration: 3000 })
.circle(id, extraCss)  /  .round(id, radius)
.circle() makes an element perfectly circular. .round() sets a specific border-radius value.
board.circle('avatar', 'border:3px solid #fff')
board.round('card', 16)
.merge(id, parent)  /  .unmerge(id)
Move elements inside a parent so they transform together (useful for rotating a group). Merged children can still be controlled individually. .unmerge() returns them to the stage root.
board.merge('item0 item1 item2', 'container')
board.rotate('container', 90, { duration: 800 })
board.unmerge('item0 item1 item2')
.sound(src, opts)  /  .stopSound(src)
Play a sound file using SoundManager2. Pass no argument to .stopSound() to stop all playing sounds.
ParameterDefaultDescription
src—Path to the MP3 file.
opts.wait optionalfalsePause timeline until the sound finishes playing.
board.sound('audio/intro.mp3', { wait: true })
board.stopSound() // stop all sounds
SoundManager2 must be loaded (soundmanager2.js) to use .sound(). If it is absent the command is silently skipped. On iOS, audio can only be triggered by direct user interaction.
.seek(index)
Jump to a specific command index and continue from there. Commands are indexed from 0 in the order they were added.
board
  .text('A', { y: 100, class: 'title center' }) // index 0
  .text('B', { y: 100, class: 'title center' }) // index 1
  .seek(0)                                          // loop back to A
  .go()
.pause()  /  .resume()
Pause and resume timeline playback. Any running animations complete before the timeline stops.
.reset()
Stop the board, clear all elements, and reset the command queue. Always call this at the start of a new presentation function before chaining new commands.
function showSlide2() {
  board.reset()
    .background('#111')
    .text('Slide 2', { y: 160, class: 'title center', show: 'fade' })
    .go();
}
.go(onDone)
Preload assets and start playback. Optionally pass a callback that fires when the last command finishes.
board.go(() => console.log('done'))

Targeting multiple elements

Any command that accepts an id also accepts a space-separated list of IDs. The command is applied to each element in order:

board
  .remove('title subtitle logo')
  .hover('btn1 btn2 btn3', 'scale:1.2', { duration: 200 })
  .fadeOut('overlay mask', { duration: 300 })

Controlling timeline flow with wait

By default each command fires and the timeline immediately advances to the next one. Set wait: true to make the timeline hold until that command fully completes:

board
  .text('First', { y: 100, class: 'title center', show: 'fade', duration: 800 }) // fires and moves on immediately
  .text('Second', { y: 200, class: 'title center', show: 'fade', duration: 800, wait: true }) // waits before continuing
  .delay(500)
  .text('Third', { y: 300, class: 'title center', show: 'fade', duration: 800 })

Built-in CSS classes

These classes are defined in styles.css and are ready to use in .text() calls:

ClassDescription
titleLarge bold white text, suitable for headings.
descriptionNormal weight body text.
description arRTL body text (Arabic / Hebrew).
centerFull-width centered text. Combine with title or description.
ButtonStyled clickable button.

Full example

const board = new BoardJS('TheShow');

function runPresentation() {
  board.reset()
    .background('rgb(20, 40, 80)', { duration: 400 })

    .text('BoardJS', { x: 0, y: 80, class: 'title center', show: 'fade', duration: 1000, wait: true })
    .text('Timeline-based animation, made simple.', {
      x: 0, y: 160, class: 'description center', show: 'fade fromleft', duration: 800, wait: true,
    })

    .image('files/logo.png', { x: 290, y: 240, show: 'fade zoom', duration: 600, easing: 'backout', id: 'logo', wait: true })
    .hover('logo', 'scale:1.1; rotate:5', { duration: 300, easing: 'backout' })

    .text('Start', { x: 300, y: 380, class: 'Button', show: 'fade', duration: 400, id: 'startBtn' })
    .on('startBtn', 'click', runNextSlide)

    .go();
}
